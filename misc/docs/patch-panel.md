# Patch Panel

## VLAN Color Code

Using color of ethernet connector cover to distinguish VLANs.
This color scheme should also match the network diagram.
Hopefully I keep a consistent white ethernet cable...we'll see.

| Color  | VLAN Name  | VLAN Tag |
| ------ | ---------- | -------- |
| White  | LAN        | 0        |
| Black  | Management | 10       |
| Green  | Home       | 20       |
| Blue   | Services   | 30       |
| Red    | Torrents   | 35       |
| Orange | Lab        | 40       |
| Yellow | IoT        | 50       |
| Grey   |            |          |
| Violet |            |          |
