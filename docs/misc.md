# Misc

## Useful Commands

- sanity check open ports - `sudo nmap -sS EDGE_ROUTER_GATEWAY_IP`
- check eth port speed - `sudo ethtool eth0 | grep -i speed`
- monitor CPU usage - `top -aSH`
